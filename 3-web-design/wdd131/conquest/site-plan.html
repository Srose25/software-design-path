Overview

Purpose

The purpose of this website is to have a playable game in it. It will be a simple point and click shooting game, but will have a settings page where you can alter the game play and look of the website.

Audience

This feels more like an extension of the blog that I did for my previous final for WDD 130. I wanted to add a small game then. The website is then for people who just want to sit back and relax a minute with an ad free arcade game.

Dynamic elements

The game will be run with JavaScript elements, and I will also be using JavaScript to adjust settings for the website.

Content

Home page

A home page for the game, giving the user the option to start the game, or adjust the settings.

Game Page

Here is where the game will be. It will have simple structure with a header nav, and a footer with socials. In the main part of the website, there will be a dropdown description of the game to the right, and an expandable option to the left, leading to other games. In the center there will be a small window for the game with the option for fullscreen. This is to help with accessibility and window size.

The game loop is simple. You have a small ship that you control, and asteroids will come at you from the top of the screen. There are three different sizes of asteroid: Small, Medium, and Large. Large rocks break into medium rocks, medium rocks break into small rocks, small rocks will disappear. The difficulty of the game will get progressively more difficult the longer you are alive. If you get hit by an asteroid then it's game over.

Settings

This page is meant to adjust settings for both the game and the way the page works as a whole. The Settings will include: Music Volume, Sound Effect Volume, Game Difficulty, Theme (adjustable across all pages), Music Choice.
